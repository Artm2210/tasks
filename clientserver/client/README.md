# Клиент (client)

SPA на **React** и **Vite** с TypeScript.

## Перед запуском

1. Установите **[Node.js](https://nodejs.org/)** (рекомендуется LTS, например 20.x).
2. Из **корня монорепозитория** `clientserver/` установите зависимости workspaces:

```bash
npm install
```

(Если открыта только папка `client/`, выполните `npm install` внутри неё один раз после клонирования.)

## Запуск и проверки

Запуск из каталога `client/`:

| Команда                | Назначение                          |
| ---------------------- | ----------------------------------- |
| `npm run dev`          | Локальный dev-сервер Vite           |
| `npm run build`        | Production-сборка                   |
| `npm run preview`      | Локальный просмотр собранной версии |
| `npm run lint`         | Проверка кода ESLint                |
| `npm run format`       | Автоформатирование Prettier         |
| `npm run format:check` | Проверка форматирования без изменений |

Из каталога `clientserver/` (корень workspaces):

| Команда             | Назначение                                              |
| ------------------- | ------------------------------------------------------- |
| `npm run client:dev` | То же, что `npm run dev` в пакете `client` (через `-w`) |

После изменений имеет смысл по очереди запускать **`npm run build`**, **`npm run lint`** и при необходимости **`npm run format:check`**.

## Как работать в проекте

- **`src/`** — приложение React: точка монтирования — `main.tsx`, корневой компонент — `App.tsx`, глобальные стили — `index.css`.
- **`index.html`** — HTML-шаблон Vite для корня страницы.
