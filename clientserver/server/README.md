# Сервер (server)

HTTP API на **Express** с TypeScript; тесты — **Vitest**.

## Перед запуском

1. Установите **[Node.js](https://nodejs.org/)** (рекомендуется LTS, например 20.x).
2. Из **корня монорепозитория** `clientserver/` установите зависимости workspaces:

```bash
npm install
```

(Если открыта только папка `server/`, выполните `npm install` внутри неё один раз после клонирования.)

## Запуск и проверки

Запуск из каталога `server/`:

| Команда                | Назначение                                                                |
| ---------------------- | ------------------------------------------------------------------------- |
| `npm run dev`          | Запуск с перезапуском при изменениях (`tsx watch src/index.ts`)           |
| `npm run build`        | Компиляция TypeScript в `dist/`                                           |
| `npm run start`        | Запуск собранного приложения (`node dist/index.js`, нужен предварительный `build`) |
| `npm test`             | Одиночный прогон тестов из `tests/`                                       |
| `npm run test:watch`   | Vitest в режиме наблюдения                                               |
| `npm run lint`         | Проверка кода ESLint                                                      |
| `npm run format`       | Автоформатирование Prettier                                               |
| `npm run format:check` | Проверка форматирования без изменений                                     |

Из каталога `clientserver/` (корень workspaces):

| Команда              | Назначение                                              |
| -------------------- | ------------------------------------------------------- |
| `npm run server:dev` | То же, что `npm run dev` в пакете `server` (через `-w`) |
| `npm test`           | Запуск тестов сервера (скрипт корня нацелен на `server`) |

По умолчанию сервер слушает порт **3001** (или значение переменной окружения **`PORT`**).

После изменений имеет смысл по очереди запускать **`npm run build`**, **`npm test`** и при необходимости **`npm run lint`**.

## Как работать в проекте

- **`src/index.ts`** — точка входа: создание приложения и `listen`.
- **`src/app.ts`** — сборка Express-приложения (`createApp`); логику маршрутов и middleware развивайте здесь.
- **`tests/`** — тесты Vitest; импортируйте модули из `src/` (как в `tests/app.test.ts`).
